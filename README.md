## Simple hand tracking application
Tracks 21 points on the hand. Each finger is a different colour, as well as the landmarks on the hand. The default number of hands detected is two but can be changed in main.py.


### INSTALLATION:


<br/>

__Linux/macOS:__


python3 -m venv .venv

source .venv/bin/activate

pip install -r requirements.txt

python3 main.py

<br/>

__Windows: (not tested)__

venv\Scripts\activate.bat

pip install -r requirements.txt

python3 main.py

<br/>

### USAGE:

press q when in the application or ctrl+c in the terminal to quit
